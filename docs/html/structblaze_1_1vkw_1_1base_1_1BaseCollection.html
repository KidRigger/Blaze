Detailed Description

template<typename THandle>
struct blaze::vkw::base::BaseCollection< THandle >

The basic collection wrapper for uniformity.

BaseCollection is only used for generating unmanaged wrapper around a collection of handles.

Template Parameters
THandleThe type of handle to own.
