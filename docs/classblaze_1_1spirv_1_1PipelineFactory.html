Detailed Description

Factory class for pipelines based on shader reflection.

PipelineFactory is created such that it will be easy to provide configuration of a pipeline and the shaders and let the factory handle matching of descriptors etc.

This makes it much easier to create pipeline layouts and allows shaders to change their descriptor set indices mostly without needing any change in the source code.
